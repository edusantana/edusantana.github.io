Criei este site pois preciso de um local para registrar informações
que não desejo perder. Você quer aprender como fazer um para você?

Eu sempre tive a necessidade de registrar diversas informações, ao me deparar com [o artigo 
Tools of The Effective Developer: Personal Logs](http://www.hans-eric.com/2007/08/31/tools-of-the-effective-developer-personal-logs) me retornou a ideia de criar um local para realizá-lo. 

As páginas do github pareceu uma solução ótima
para manter estes registros, principalmente porque tem uma
aprecencia de blog e podemos escrever as mensagens off-line.

Quer fazer um para você? Consulte [ajuda do github](https://help.github.com/categories/20/articles) ou acesse [o repositório deste site](https://github.com/edusantana/edusantana.github.io).

As páginas do github utilizam Jekyll, para saber mais consulte também:

* http://jekyllbootstrap.com/lessons/jekyll-introduction.html
* http://net.tutsplus.com/tutorials/other/building-static-sites-with-jekyll/
